## Requirements

- MongoDB 3.6+

- pymongo 3.6+

## Usage

### MongoDB Scripts

```bash
# List all databases
./0-list_databases

# Create/use database
./1-use_or_create_database

# Insert document
./2-insert

# List all documents
./3-all

# Find documents
./4-match

# Count documents
./5-count

# Update documents
./6-update

# Delete documents
./7-delete

# Find with regex
./100-find
```

### Python Scripts

```bash
# List all documents
python3 8-all.py

# Insert document
python3 9-insert_school.py

# Update topics
python3 10-update_topics.py

# Find by topic
python3 11-schools_by_topic.py

# Log stats
python3 12-log_stats.py
``` 
